 Project Overview

This project is part of the Digital Literacy course at VIT Bhopal University.
As a Student Digital Ambassador, the objective of this project is to help peers understand and effectively use digital tools, maintain a professional online presence, and stay safe in the digital world.

The project is divided into five tasks, each focusing on a different aspect of digital literacy.
